# 데이터 출력하고 정렬하는 법 (SQL SELECT, ORDER BY)

SQL: Structured Query Language, 데이터베이스 조작에 필요한 언어

SQL로

- 데이터 삽입
- 데이터 출력
- 데이터 삭제
- 데이터 수정
  가능

깊게 들어가면 조건문, 반복문, 함수, 변수 등의 프로그래밍 가능

## SELECT FROM

- 데이터를 출력하는 문법

```
select * from 테이블명
```

=> 해당 테이블의 모든 데이터 출력

```
select 컬럼명 from 테이블명
```

=> 해당 테이블에 있는 특정 컬럼에 해당하는 모든 데이터 출력

```
select 상품명, 가격 from 테이블명
```

=> 해당 테이블에 있는 상품명과 가격 컬럼에 해당하는 모든 데이터 출력

## ORDER BY

데이터 출력만 하는 것이 중요한게 아니라
가공할 줄 알아야함

## 출력한 데이터 정렬하는 법

```
select 컬럼명/* from 테이블명 order by 컬럼명
```

### 예시

예시:

```
select * from `fake products` fp
```

결과:

```
1	셔츠	상의	3000
2	반팔티	상의	4000
3	침대	가구	5000
4	의자	가구	6000
5	하이탑	신발	3000
```

예시:

```
select * from `fake products` fp order by 가격
```

결과:

```
1	셔츠	상의	3000
5	하이탑	신발	3000
2	반팔티	상의	4000
3	침대	가구	5000
4	의자	가구	6000
```

## 오름차순/내림차순(ascending/descending)

```
select 컬럼명/* from 테이블명 order by 컬럼명 asc/desc
```

기본값은 asc

### 예시

예시:

```
select * from `fake products` fp order by 가격 asc
```

결과:

```
1	셔츠	상의	3000
5	하이탑	신발	3000
2	반팔티	상의	4000
3	침대	가구	5000
4	의자	가구	6000
```

예시:

```
select * from `fake products` fp order by 가격 desc
```

결과:

```
4	의자	가구	6000
3	침대	가구	5000
2	반팔티	상의	4000
1	셔츠	상의	3000
5	하이탑	신발	3000
```

### 카테고리순으로 정렬 후에 2차로 가격순으로 정렬하고 싶다면

예시:

```
select * from `fake products` fp order by 카테고리 asc, 가격 desc
```

결과:

```
4	의자	가구	6000
3	침대	가구	5000
2	반팔티	상의	4000
1	셔츠	상의	3000
5	하이탑	신발	3000
```

'카테고리' 기준으로 오름차순(asc) 먼저 한 다음 같은 '카테고리'가 있으면 내림차순(desc)으로 정렬

## 문제

### Q1. 현재 테이블의 상품명과 가격컬럼 출력

### Q2. 현재 테이블의 모든 컬럼을 출력하는데 가격 낮은 순으로 정렬

### Q3. 현재 테이블의 모든 컬럼을 출력하는데 가격 낮은 순으로, 가격이 같으면 카테고리가 가나다순으로 뜨도록 정렬

### Q4. 근무기간이 적은 직원은 맨 위에, 같은 근무기간인 경우 이름을 가나다 순으로 정렬
